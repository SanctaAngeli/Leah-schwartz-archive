# Leah Schwartz Digital Archive

A digital museum experience honoring the artistic legacy of Leah Schwartz (1945–2004).

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Status

**Live** at [leah-schwartz-archive.netlify.app](https://leah-schwartz-archive.netlify.app) — real artwork throughout,
built out from the 1990 Strawberry Press monograph. The lobby, the five wings,
the timeline, and the Atlas are all in place.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Deployed on Netlify

## Design Philosophy

- Gallery white backgrounds — the art brings the texture
- Glassmorphism UI — frosted glass, soft shadows, floating elements
- Organic animations — everything breathes, nothing static
- Premium feel — Apple-level design quality
