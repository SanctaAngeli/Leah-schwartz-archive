# Textures

Drop paper / deckle / wash background textures here. Anything in this folder is
served at `/textures/<filename>` and can be layered behind paintings on the
front page and elsewhere (the way the painting sits on a paper ground in the
imagined homepage mock).

Suggested naming so they're easy to wire up:
- `paper-cream.jpg` — base warm paper ground
- `deckle-edge.png` — transparent torn-edge overlay/mask
- `wash-soft.jpg` — faint watercolor wash

Tell me which file should sit where and I'll wire them into the front page
(replacing the current SVG-turbulence deckle approximation with the real thing).
